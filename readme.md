# Проекты с Qt:
- ## TaskManager – Менеджер задач с графическим интерфейсом
(десктопное приложение для управления задачами, разработанное на C++ с использованием фреймворка Qt. Программа предоставляет удобный интерфейс для создания, отслеживания и организации задач с установкой сроков выполнения и приоритетов. Приложение демонстрирует практические навыки разработки кроссплатформенного ПО с современными подходами к архитектуре).
